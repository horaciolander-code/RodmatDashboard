# Rodmat Dashboard V2

SaaS multi-tienda para gestión de inventario y ventas TikTok Shop.

## Stack
- **Backend:** FastAPI + SQLAlchemy + SQLite
- **Setup Panel:** React (Fase 3)
- **Dashboard:** Streamlit (Fase 5)

## Quick Start

```bash
cd backend
pip install -r requirements.txt
python run.py
```

Swagger UI: http://localhost:8000/docs
